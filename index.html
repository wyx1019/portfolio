Haco
- 完全定額制住宅

制作： 個人制作（企画、デザイン、一部の撮影、コーディング）
制作時期： 主に2019年11月-1月、2020年10月にデザインとレスポンシブ対応の修正を入れた
使用スキル： JQuery/JavaScript、 SASS/CSS、 HTML、 Google Map api
対応デバイス： PC・タブレット・スマホ対応
サイトを見る　

背景説明

　　背景： 入学 1 年目後期の課題です。企画・デザイン・コーディングはすべて個人制作です。 クライアントから完全定額制住宅「Haco」のランディングページ制作という依頼を受け て作ったものです。 「Haco」とは Hayama Code という意味を持つブランド名で、葉山で家を建てる完全定額 制プランです。家の雰囲気はラフカジュアルなブルックリンスタイルに決まっています。 サイトの目的は葉山と「Haco」の認知度を上げて、お問い合わせの数または公式サイト への流入数を増やすことです。

※Google Map apiを使ってみました

2020年6月にGoogle Map APIを使用して、葉山のマッピングを追加しました。 マップのスタイルのカストマイズ、表示項目の変更、複数のマーカーと吹き出しの設置などをやってみました。

　　大学院で使った統計解析ソフトが私はプログラミングに好奇心を持ったきっかけです。それに加えて、IT技術で世の中が変化し続くことに気づき、新しい価値を創造する仕事にやりがいを感じました。将来プログラミングを身につけ、技術を活かせる仕事をしたいと思いました。web を選んだ理由の一つは成果がすぐにビジュアルで見れて、達成感を感じやすいこと。また、デザイナーの知人からの影響で、デザインに憧れる部分もあります。今後、技術を中心に高めていくながら、デザイン力も身につけたらという考えを踏まえて、私は専門学校でWebデザインを勉強することを決めました。

　　将来、Webエンジニアを目指して自分のスキルを高めたいと思います。モダンな技術を追いかける上、ユーザビリティを考えた便利なサービスを世の中に届けたいです。
　　技術を高めると共に、語学の勉強も日々忘れずにやっていきます。今後日本で生活する上、いち早く日本人並みの日本語を身につけたいと強く願っております。将来の仕事の中、もし英語・中国語を活かせる機会があれば、力を発揮できる人間になりたい。
　　また、私は、2018年から結婚しており、現在子供なし。将来日本に住む予定をしております。5年以内に出産・引っ越しの予定はありません。しばらくの間に、自分のキャリヤを築き上げたいと思います。

　　旅行が大好きです。日本国内ですでにたくさんの旅をしました。旅先で発見された歴史・文化がいつも私をワクワクさせます。観光客の多い場所より、主人と一緒に秘境に訪ねることが好きです。私にとって，旅行は頭のリフレッシュと新しいものを勉強するチャンスです。
　　最近ハマったことは、歴史・芸術です。旅先の美術館が私の必ず行く場所となっております。今年（2020年）、在宅時間が多くなっているので、ドキュメンタリーを見たり、ヨーロッパ史や西洋美術史に関する本を読んだりしております。それゆえに、ヨーロッパへ遊びに行きたくなりますね！

　　私はコツコツと努力を積み重ね、自分のスキルを伸ばすことに自信があります。この長所を最も発揮したのが、大学時代の日本語独学です。大学の専攻は英語でしたが、日本文化にハマったことで、日本語を学んで、将来日本留学に行くと考えていました。忙しい大学生活の中、学習を継続的に進められるように、私はしっかりと自分のスケジュールを管理していました。毎日必ず日本語のコンテンツを聞くほか、毎週しっかり勉強する時間が確保できることを意識してスケジュールを立てました。結果として、私は2年半で0から日本語能力試験N1に合格することを達成出来ました。
　　日本に来てから、私は英語の勉強もやめていません。毎日NHK worldを聞くことを習慣として身につけております。毎日継続的な努力で、私は2018年にTOEICテストための対策がほとんどなしで890点を取りました。        
　　そのほか、私は好奇心が強く、新しいことを勉強するのが好きです。気になることであれば、常にチャレンジを恐れず自ら学んでいきます。コーディングを勉強する際に、知らない技術や言葉とたくさん出会い、私はいつも調べずにいられなく、理解を深めるために努力しております。過去の作品にRuby on Railsを取り入れることも、好奇心に駆られた結果です。
　　Web業界の技術更新が速いというところは、私にとって、新しいことに出会え、学び続ける良い環境だと思っております。将来も、努力を積み重ね、学び続ける姿勢で自分の技術を高めていきたいと思います。
