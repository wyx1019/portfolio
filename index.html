スポフェス
- Campus Tweets ver.

制作： 個人制作（企画、デザイン、コーディング）
制作時期： 2019年5月-7月、2020年7月-8月
使用スキル： Ruby 2.6.6（独学）、 Rails 6（独学）、 Bootstrap（独学）、 JQuery、 SASS、 HTML

背景説明

　　Ruby on Railsでコミュニティ機能を追加した2作目のspofesサイトです。
元々は入学1年目前期の課題です。毎年の夏に開催されるスポーツフェスティバル、つまり学校の運動会のサイトです。 全校生徒にスポフェスを楽しみにしてもらうことを目的にして制作しました。

機能

  • ユーザーログイン・登録機能
  • 投稿（画像アップロード可能）・コメント・返信機能
  • いいね機能
  • フォロー機能
  • 検索機能
  • キーワード&入学年度でユーザーを検索する
  • 過去年度の投稿を見る（テストのため、現在は７－９月分の選択肢となっております。）
  • 入力データのバリエーション
  • adminユーザーだけが競技種目を追加・更新・削除することができる
  • ユーザーが競技ページの参加ボタンを押すことによって、参加している競技種目が他のユーザーに分かってもらう。競技種目の詳細を見ると、参加しているユーザーの名前も表示されております。
  • ログインユーザーと非ログインユーザーの使える機能が制限されております
  • 開始時間までのカウントダウン（テストのため、開始時間の設定は2020.10.19 10:00　a.m.となっております。）

Haco
- 完全定額制住宅

制作： 個人制作（企画、デザイン、コーディング）
制作時期： 2019年11月-1月
使用スキル： JQuery、 SASS、 HTML
サイトを見る　

背景説明

　　背景： 入学 1 年目後期の課題です。企画・デザイン・コーディングはすべて個人制作です。 クライアントから完全定額制住宅「Haco」のランディングページ制作という依頼を受け て作ったものです。 「Haco」とは Hayama Code という意味を持つブランド名で、葉山で家を建てる完全定額 制プランです。家の雰囲気はラフカジュアルなブルックリンスタイルに決まっています。 サイトの目的は葉山と「Haco」の認知度を上げて、お問い合わせの数または公式サイト への流入数を増やすことです。
